# ARES Clan Hub

Прототип закрытого цифрового мира клана ARES: регистрация, личный кабинет, профиль, чат, публикации, магазин и административная панель.

## Запуск

1. Перейдите в директорию проекта:
   ```bash
   cd /workspaces/Ares
   ```
2. Установите зависимости:
   ```bash
   npm install
   ```
3. Запустите сервер:
   ```bash
   npm start
   ```
4. Откройте в браузере:
   ```
   http://localhost:4000
   ```
