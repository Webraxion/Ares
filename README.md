# ARES Clan Hub (Frontend Only)

Статический сайт клана ARES на HTML, CSS и JavaScript с **шифрованным хранилищем данных на GitHub**.

## Как запустить

1. Откройте `public/index.html` в браузере.
2. Используйте форму входа или регистрацию.

## Особенности

- Регистрация и уникальный ID игрока
- Личный кабинет с настройкой профиля
- Темный стиль ARES, фон танков и эффект огня
- Общий клановый чат
- Личные сообщения по ID или через друзей
- Система друзей и заявки в друзья
- Магазин, внутренняя валюта и покупки
- Лента публикаций, лайки и комментарии
- Панель администратора / куратора
- Достижения и уведомления

## Синхронизация данных через GitHub

### Как подключить:

1. **Создайте GitHub Personal Access Token**:
   - Перейдите на https://github.com/settings/tokens
   - Нажмите "Generate new token (classic)"
   - Выберите scope: `gist`
   - Скопируйте токен (вид: `ghp_xxxxx`)

2. **Подключите в ARES**:
   - На главной странице (`index.html`) нажмите "Синхронизация через GitHub"
   - Введите GitHub токен и пароль для шифрования
   - Нажмите "Подключить GitHub"

3. **Сохраняйте и загружайте данные**:
   - На панели (dashboard) есть кнопки "💾 GitHub" и "📥 GitHub"
   - Или в профиле ("Синхронизация данных")

### Как это работает:

- Данные шифруются **AES-256-GCM** перед отправкой на GitHub
- Сохраняются в приватный GitHub Gist
- Только вы можете их расшифровать (нужен пароль)
- Локальное хранилище (`localStorage`) по-прежнему используется
